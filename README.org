** Global Variables
*** setq
+ [(setq <var> <value>)]
+ [(setq <var1> <value1> <var2> <value2> ...)]
+ set one or more varibales. return the last value
#+BEGIN_SRC emacs-lisp
  (setq x 3)
  (setq x1 1 x2 2 x3 3)
  (message "x1=%d x2=%d x3=%d" x1 x2 x3)
#+END_SRC

#+RESULTS:
: x1=1 x2=2 x3=3

#+END_SRC

#+RESULTS:

** Local Variables
*** let
+ [(let (<var1> <var2> ...) <body>)]
+ [(let VARLIST BODY)]
+ The body is one or more lisp expressions. The body's last expression's value is returned.
#+BEGIN_SRC emacs-lisp
(let (a b)
 (setq a 3)
 (setq b 4)
 (+ a b)
) ;  7
#+END_SRC

#+BEGIN_SRC emacs-lisp
  (let ((a 3) (b 4))
    (+ a b))
#+END_SRC

#+RESULTS:
: 7

* Block of Expression
** progn
when you need to group several expression together as one single expression
this can be done with [progn] , it will return the last expression 
#+BEGIN_SRC emacs-lisp
  (progn
    (message "a")
    (message "b"))

  (progn
    (setq x 1)
    (setq y 2)
  (+ x y)
  ;;similar to sequential control construct , programm in the progn will be executed sequentially , like C {}
#+END_SRC

#+RESULTS:
: b
** prog1 , prog2
similar to progn , prog1 return the first expression , prog2 return the second expression
#+BEGIN_SRC emacs-lisp

    (message "b"))
#+END_SRC

#+RESULTS:
: a
#+BEGIN_SRC emacs-lisp

+ campare to if expression , when expression is not else part

  #+END_SRC

** cond expression
+ [(cond (<condition1> <body1>) (<condition2> <body2>) ... (t <body>)]
+ like switch
#+BEGIN_SRC emacs-lisp
(setq ux 11)
  (cond
    ((eq ux 10) (message "ux = %d" ux))
    ((not (eq ux 10)) (message "ux not 10 , ux = %d" ux))
  (t (message "not ux")))
#+END_SRC
